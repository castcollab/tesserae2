# pycortex

Python implementation of [CortexJDK](https://github.com/mcveanlab/CortexJDK)

Pycortex supports python versions 3.5 and 3.6.

## Development

Set up the project

```bash
make init
```

See `pycortex/__main__.py` for example uses of pycortex.
