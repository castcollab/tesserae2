# cortexpy

Python implementation of [CortexJDK](https://github.com/mcveanlab/CortexJDK)

Cortexpy supports python versions 3.5 and 3.6.

## Development

Set up the project

```bash
make init
```

See `cortexpy/__main__.py` for example uses of cortexpy.
